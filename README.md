# fm-rank-calc
The Official Japanese Guide contains a section going over how your duel rank is calculated. By cross referencing the English version with the Japanese version, I have determined the following data.

The following is the scale of how many "points" you need for each rank.

99-90: S POW

89-80: A POW

79-70: B POW



69-60: C POW

59-50: D POW

49-40: D TEC

39-30: C TEC

29-20: B TEC

19-10: A TEC

09-00: S TEC

Basically, you start off the duel at 50 points. From there, you add or subtract the following to figure out your point total.
